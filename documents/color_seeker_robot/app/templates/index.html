Color Target Seeker Robot
\n
\n Welcome to the control console for the Autonomous Color Target Seeker Car. This system integrates a two-wheeled differential chassis with ROS2 node communications, computer vision target locks based on HSV thresholding, and a secure Flask telemetry panel.\n
\n\n
\n
\n
Target Color Selection
\n

\n Allows configuring target search properties. Scans and locks onto specific hue filters (Red, Green, Blue, or Yellow cylinders/cards).\n

\n
\n
\n
Rotational Scan Search
\n

\n Triggers rotational search patterns (spin commands) until the target hue coordinates cross the camera central axis threshold.\n

\n
\n
\n
Distance Proximity Stop
\n

\n Calculates relative size and distance of target from camera, steering to approach the target and bringing the robot to a controlled stop at exactly 10 cm.\n

\n
\n
\n
Secure Control Bridge
\n

\n Requires authorized Operator login. Provides real-time log monitoring, database tracking of seeker history, and E-STOP overrides.\n
